Utilisation

Dans le menu de navigation de l’administration, il y a un lien Taxonomies. Cliquer dessus pour aller vers la liste des taxonomies. Depuis cette page il est possible de créer, modifier ou supprimer des taxonomies. Il est aussi possible de lister les termes de taxonomie, puis de créer, modifier ou supprimer des termes pour cette taxonomie.

Une fois que les taxonomies sont créées, créer/modifier un modèle de ressource et inclure les nouveaux types de données pour les propriétés souhaitées. Utiliser ce modèle de ressource lorsque les taxonomies sont nécessaires.

Remplacer des valeurs existantes par des termes de taxonomie

Il est possible de remplacer des valeurs littérales existantes par des liens vers des termes de taxonomie en utilisant l’outil de modification par lot:

  • Chercher des contenus, collections ou médias,

  • Sélectionner Modifier tout ou Modifier sélectionnés dans la liste déroulante Traitements en lot,

  • Cliquer sur Traiter

  • Dans Remplacer les valeurs littérales par des termes de taxonomie (propriétés) sélectionner les propriétés à modifier,

  • Dans Remplacer les valeurs littérales par des termes de taxonomie (taxonomie), sélectionner la taxonomie contenant les termes souhaités,

  • Cliquer sur le bouton Enregistrer.
