Bloc de page de site

Le bloc de page « Arborescence de termes de taxonomie » affiche tous les termes racine d’une taxonomie et permet aux utilisateurs de charger les enfants d’un terme en cliquant dessus. Cliquer sur le titre du terme les emmène sur la page du terme de taxonomie où ils peuvent voir les propriétés du terme de taxonomie ainsi que les ressources liées.

Le lien et l’apparence peuvent être personnalisés par les thèmes en surchargeant taxonomy/common/taxonomy-term-jstree-node-text-public.phtml

Pour ajouter le bloc « Arborescence de termes de taxonomie » à une page, voir la documentation officielle sur comment modifier les pages.

Une fois le bloc ajouté à une page vous avez la possibilité de sélectionner une taxonomie. Seuls les termes appartenant à cette taxonomie seront affichés.
